# Iconic Project Tracker - Minimal Version

**Status: Phase 1 - Basic Working App** ✅

## Quick Deploy

### 1. Upload to GitHub
- Create new repo: `iconic-tracker`
- Upload all files

### 2. Deploy to Vercel
- Import from GitHub
- Deploy (auto-detects Next.js)

### 3. Add Environment Variable
```
Name: NEXT_PUBLIC_DRIVE_JSON_URL
Value: https://drive.google.com/uc?export=download&id=YOUR_FILE_ID
```

### 4. Promote to Production
Done! App is live.
